# • English School Telegram Bot

> Telegram-бот для автоматизации записи на занятия английским языком.  
> Разработан на Python с использованием Aiogram.

---

# ■ Возможности бота

## • Пользовательский функционал

✔ Запись на занятия  
✔ Отправка заявок через Telegram  
✔ Сбор информации о пользователе  
✔ Просмотр ближайшего группового занятия  
✔ Получение описания и медиа материалов занятия  

---

## • Административный функционал

### ■ Система прав администратора

Администратор получает доступ к специальным командам управления ботом.

### ■ Команда: /new_sc

Позволяет:
- добавить название занятия
- добавить описание
- загрузить фото или видео
- обновить информацию о ближайшем групповом занятии

### ■ Команда: /request

Позволяет:
- просматривать оставленные заявки
- получать username пользователей
- видеть контактную информацию
- обрабатывать записи учеников
